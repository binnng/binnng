# Generator


- generator-webapp
- generator-angular
- generator-require
- ...

不同的模板
每个`generator`都是npm模块

[slide]

```
// 安装 Yeoman

npm install -g yo bower grunt-cli

// 安装`Generator`
npm install -g generator-webapp

// 新建项目
mkdir demo && cd demo && yo webapp
```
